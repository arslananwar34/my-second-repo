Assign and change permissions of file and folders

r= read,    w= write,    x= execute

 Owner    groups    users
  rw-       r-x      r--

owner change of file and folder : chown (username) (filename)
                                  chown ali dir1
                                  chown ali dir2

change group of file and folder : chgrp (gropname) (foldername)
                                  chgrp dev-grp dir2
